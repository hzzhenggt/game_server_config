# simple server config manager  | 简易游戏配置管理器
support linux/mac ssh  file compare deploy   | 支持linux/mac的ssh 管理文件，
don't use it in production | 请不要用于生产环境！！！

### requirements | 依赖  
django==3.2.5  
django-cors-headers==3.7.0  
paramiko  
django-bootstrap5==22.2  
chardet
